# AGENTS.md

## Prosjektmål
Dette prosjektet er en profesjonell håndbok på norsk for nye medarbeidere som jobber med marine reservedeler, motorserienummer, bestilling av deler og kundesupport.

## Målgruppe
- Nye ansatte i salg, support, ordrebehandling og reservedelsservice
- Personer som håndterer marine motorer og reservedeler
- Ansatte som skal forstå arbeidsflyt, dokumentasjon og kundekommunikasjon

## Innholdsprinsipper
- Skriv på norsk, ryddig og profesjonelt
- Vær presis, praktisk og arbeidsorientert
- Fokus på faktisk arbeid i praksis med reservedeler og motorer
- Inkluder relevante merker og modeller:
  - Caterpillar
  - Volvo Penta
  - Mitsubishi
  - Cummins
  - MAN
  - Scania
  - Yanmar

## Struktur
- Bruk mappen chapters for hovedinnhold
- Bruk glossary for fagord og forklaringer
- Bruk checklists for kontrollister og arbeidsflyter
- Bruk quiz for repetisjon og læring
- Bruk output for ferdige dokumenter eller eksporterte versjoner

## Arbeidsretningslinjer
- Hold kapitlene kortfattet, men informative
- Inkluder praktiske eksempler og typiske feil som kan forekomme
- Vær tydelig på hvordan serienummer og delnumre brukes
- Legg vekt på korrekt dokumentasjon og kundeservice
- Oppdater innholdet regelmessig ved endringer i produkter eller prosesser
