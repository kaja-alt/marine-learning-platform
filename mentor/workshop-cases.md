# 150 realistiske verkstedscaser

## VC-001 - Caterpillar (CAT) / kjølesystem
**Symptom:** Caterpillar (CAT) i marine installasjon viser høy temperatur.
**Observasjoner:**
- saltkrystaller rundt slangeklemme
- nylig utført service
- alarm lagret i display
**Feilsøkingsprosess:**
- Sikre motoren og avklar om videre kjøring kan gi følgeskade.
- Kontroller motoridentitet: ESN, Serial Number Prefix og Arrangement Number.
- Bekreft observasjoner i kjølesystem med bilder, måling eller fysisk inspeksjon.
- Kontroller sjøvannsfilter, impeller, varmeveksler og termostat før komponenten dømmes defekt.
- Slå opp aktuelle deler i SIS / Parts Manual og dokumenter delnummer, variant og supersession.
**Aktuelle komponenter:**
- impeller
- sjøvannsfilter
- varmeveksler og termostat
- pakninger/kit
- sensor eller kontakt der relevant
**Sannsynlig årsak:** Mest sannsynlig årsak er avvik i kjølesystem, ofte knyttet til sjøvannsfilter, impeller, varmeveksler og termostat, men endelig konklusjon krever måling og produsentdata.
**Anbefalt reparasjon:** Anbefalt reparasjon er å utbedre dokumentert årsak, ikke bare bytte impeller. Bestill riktig del eller kit etter kontroll i SIS / Parts Manual, skift tilhørende pakninger/slitedeler og kjør funksjonstest etter montering.
