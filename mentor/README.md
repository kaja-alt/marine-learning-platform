# Mentor for nyansatt innen marine reservedeler

Denne mappen er et praksisorientert mentorlag som brukes sammen med håndboken og den digitale plattformen. Innholdet trener nyansatte i å tenke som en erfaren reservedelsselger, teknisk supportmedarbeider og verkstedkoordinator.

## Innhold
- `customer-cases.md` og `customer-cases.json`: 150 realistiske kundecaser.
- `workshop-cases.md` og `workshop-cases.json`: 150 realistiske verkstedscaser.
- `mentor-guide.md`: forslag til hvordan casene brukes i opplæring.

## Bruk
Velg 3-5 caser per uke. La deltakeren først skrive eget kundesvar eller feilsøkingsnotat, og sammenlign deretter med korrekt løsning. Alle produsentspesifikke data skal verifiseres mot offisiell dokumentasjon før bruk i reell kundeordre.
