# Marine Handbook

Profesjonell opplæringshåndbok på norsk for nye medarbeidere hos marinemotorforhandler, med fokus på marine dieselmotorer, reservedeler, serienummer, motornummer, delnummer, feilsøking, vedlikehold, ordrebehandling og kundesupport.

Håndboken dekker Caterpillar (CAT), Volvo Penta, Mitsubishi, Cummins, MAN, Scania og Yanmar. Innholdet er skrevet som intern opplæringsmanual og skal brukes sammen med kontrollister, quiz, instruktørveiledning, studentarbeidsbok, eksamen, søkbar indeks og en digital opplæringsplattform i `website/`.

## Struktur
- `chapters/` inneholder 50 komplette kapitler.
- `glossary/` inneholder samlet norsk/engelsk ordliste.
- `checklists/` inneholder praktiske kontrollister for arbeidshverdagen.
- `quiz/` inneholder repetisjon og fasit.
- `images/` inneholder SVG-diagrammer og feilsøkingsflyter.
- `output/` inneholder samlet håndbok, instruktørmateriell, eksamen og utskriftsgrunnlag.
- `website/` inneholder digital opplæringsplattform med dashboard, progresjon, quiz, søk, hurtigoppslag, produsentoversikter og lys/mørk modus.
- `build/` inneholder PDF-byggefiler.

- `mentor/` inneholder kundecaser, verkstedscaser og mentorveiledning for nyansatte.
- `daily-training/` inneholder 365 daglige treningsoppgaver.
- `flashcards/` inneholder over 1000 spørsmål og svar for repetisjon.
- `common-mistakes/` samler vanlige nyansatt-feil og hvordan de unngås.
- `decision-trees/` inneholder beslutningstrær for sentrale arbeidsprosesser.
- `cheat-sheets/` inneholder korte A4-ark for utskrift ved arbeidsplassen.

- `website/parts-database.json` inneholder lokal opplæringsdatabase for reservedelssøk med produsentverifikasjon.
- `website/manifest.webmanifest` og `website/service-worker.js` gjør plattformen til PWA med offline-støtte.
- Plattformen har interaktive diagrammer, reservedelsdatabase og arbeidsmodus med lokal lagring.
