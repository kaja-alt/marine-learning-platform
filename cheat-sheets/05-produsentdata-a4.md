# Produsentdata - A4 hurtigark

## Bruk
Skriv ut arket og bruk det ved telefon, delelager eller verkstedmottak.

## Sjekkpunkter
- CAT: ESN, prefix, Arrangement Number
- Volvo Penta: Engine Type, Serial, Product Number
- Cummins: ESN og CPL
- MAN/Scania: engine number og rating
- Yanmar: model code og serial

## Husk
- Verifiser alltid i offisiell kilde
- Marker antakelser tydelig

## Produsentverifikasjon
Konkrete delnumre, intervaller, garantikrav og tekniske verdier skal alltid kontrolleres mot produsentens offisielle dokumentasjon.
